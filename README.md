## Типы данных

- Неизменяемые\
  a = 1\
  b = a

- Изменяемые\
  a = [1]\
  b = a\
  b.appent(1)\
  a и b смотрят на один список, поэтому меняются вместе

### Типичные варианты файлов

- .yaml
    - Конфигурационные файлы
- .csv
    - Пользовательская информация
- .txt
    - Любая информация
